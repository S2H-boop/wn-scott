# wn-scott — Scott Millward's Principal Agent

Coordination repo for Scott Millward's Principal agent at Woody Nelson.

## Setup (first time, on your Mac)

1. Install Claude Code: https://claude.ai/code
2. Clone this repo: `git clone git@github.com:fm0x2a/wn-scott.git ~/wn-scott`
3. Open Claude Code in this directory: `claude ~/wn-scott`
4. On first session: paste the onboarding prompt Frank shared with you

## Files

| File | Purpose |
|---|---|
| `CLAUDE.md` | Project instructions -- loaded automatically by Claude Code |
| `SCOTT.md` | Your communication channel with Wren and the Squad |
| `README.md` | This file |

## How it works

Claude Code reads `CLAUDE.md` automatically when you open the project. Your agent pulls `SCOTT.md` at session start to get messages from the Squad. Messages go both ways -- Wren writes to you here, you write back.

Pull at the start of every session. Push at the end. The git history is your permanent record.
